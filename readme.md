# Configuración de mi oh-my-zsh

Configuración del archivo .zshrc

### Version
0.0.1

### Instalación
El fichero .zshrc tiene que estar en el $HOME del usuario
```sh
$ ~ ls -a
.  ..  .zshrc
```
